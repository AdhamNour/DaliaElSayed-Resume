\section{Impact Highlights}

\begin{itemize}
  \item Reduced key financial pipeline processing time by \textbf{83\%} through targeted SQL
        and ETL optimization at Crédit Agricole Egypt.

  \item Improved data freshness by \textbf{60\%} via a real-time Kafka ingestion pipeline,
        enabling faster stakeholder decision-making at Al-Rajhi Bank.

  \item Cut API calls by \textbf{70\%} through integrated real-time data flows, improving
        system reliability across multiple banking systems.

  \item Projected significant annual cost savings by migrating Al-Rajhi Bank from
        Informatica/Oracle licenses to an open lakehouse (Iceberg + Trino).

  \item Boosted Trino query performance by up to \textbf{70\%} through a unified semantic
        layer consumed by BI tools across the bank.

  \item Delivered \textbf{40\%} reduction in data processing time and \textbf{25\%}
        operational cost reduction by modernizing the data platform architecture.
\end{itemize}
